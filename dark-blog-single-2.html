Product Details Scraping with Selenium and Pandas

January 20, 2025
in, LiveStyle
120
89

This project demonstrates how to scrape product details from an e-commerce website using Selenium and organize the data into a structured format with Pandas. The process involves navigating to a website, searching for a product (e.g., "Samsung"), and extracting relevant product information from the first page, such as product names, store details, and prices. The data is then compiled into a DataFrame and exported into an Excel sheet for further analysis or reporting.

  • Data Cleaning: I started by ensuring the data was clean and ready for analysis. This involved checking for missing values, duplicates, and inconsistencies. Where necessary, I performed imputation for missing values or removed unnecessary duplicates to maintain data integrity.

  • Statistical Summary: I generated summary statistics for the numerical features, including measures such as the mean, median, standard deviation, and percentiles, to provide a quick overview of the data distribution.

  • Feature Engineering: I created new features based on the relationships between existing columns, such as the Petal-to-Sepal Ratio, to reveal additional insights and enhance model-building efforts.

This project showcases my ability to handle various aspects of data exploration, from data cleaning to advanced statistical analysis and visualization. It demonstrates my proficiency in using Python to uncover insights from data and my understanding of best practices in the field of data science.
